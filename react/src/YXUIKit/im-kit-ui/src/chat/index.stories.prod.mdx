# 聊天组件

聊天组件负责用户间消息传递、消息展示、聊天控制等功能。提供了聊天输入框、聊天内容展示和控制、聊天设置等。

- 聊天输入框

  1. 发送消息
  2. 发送表情
  3. 发送图片
  4. 发送文件

- 聊天内容展示和控制

  1. 聊天消息展示、好友资料展示
  2. 聊天消息撤回、删除、重新编辑
  3. 聊天好友拉黑/解除拉黑、删除/添加

- 聊天设置

  1. 单聊设置，创建群聊
  2. 群聊设置，群解散/群退出、群资料展示与修改、群成员添加/移除、群权限设置等
