# 会话列表组件

会话列表组件负责显示当前用户单聊及群聊的会话列表。提供会话点击和右键菜单功能对会话进行操作。

- 会话点击支持功能

  1. 选中会话
  2. 重置会话未读数

- 右键菜单支持功能
  1. 会话免打扰
  2. 删除会话
